# Image storage policy

Final website photos for Discovery Sound Garden should live in Cloudflare Images, not in GitHub.

GitHub should only store tiny permanent assets such as a logo, favicon, small icons, or placeholder SVGs. Do not commit large original photos, exported social graphics, or unapproved final images.

Use lowercase hyphenated filenames or readable Cloudflare custom ID paths. Track each image in the `DSG_IMAGES` inventory in `assets/js/site-data.js`.

Recommended Cloudflare custom IDs:

- `homepage/hero-music-opportunity-v1`
- `homepage/start-learning-sheet-music-v1`
- `homepage/vanguard-voices-rehearsal-v1`
- `homepage/create-record-microphone-v1`
- `homepage/donation-opportunity-v1`
- `about/founders-v1`

Approval checklist:

- Approved for website use
- License confirmed
- Consent confirmed or not needed
- Alt text written
- Cloudflare URL tested
